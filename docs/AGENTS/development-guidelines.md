# Development Guidelines

**CRITICAL: Context7 MCP Server Usage**

- Context7 MCP server is ALWAYS available for code planning and implementation
- ALWAYS use Context7 MCP server to get optimal code practices and code examples when planning or writing code
- Use `mcp_context7_resolve-library-id` to find library IDs before getting documentation
- Use `mcp_context7_get-library-docs` to retrieve up-to-date documentation and best practices
- This ensures all code follows modern best practices and uses current patterns
